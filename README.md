## Run Application
#### install meteor
* [미티어 설치](https://www.meteor.com/install)
#### npm install (최초 1회 실행)
```
meteor npm install
```
#### run
```
meteor
```
* 실행 후 웹브라우저에서 localhost:3000 접속  (```meteor run --port <port number>```로 포트번호 지정 가능)
